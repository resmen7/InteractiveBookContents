# InteractiveBookContents

This project is a simple **table of contents manager** implemented in Java.  
Users can add headings to a list, and duplicate headings are not allowed.  
Headings are automatically added in order at the end of the list.

## Features
- Add headings to the list
- Prevent duplicate headings
- Headings are added in order
- Console-based usage

## Usage
1. Download the project
2. Run it using a Java IDE or terminal
3. Follow the console prompts to add headings

## Technologies
- Java
- Object-Oriented Programming (OOP)
- Basic data structures
